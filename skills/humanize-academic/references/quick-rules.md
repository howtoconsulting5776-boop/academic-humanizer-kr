# Quick Rules — Fast Monolith 1줄 규칙 (Academic)

`academic-humanize-monolith`가 단일 패스로 적용하는 압축 규칙. 불가침 토큰(고유명사·
숫자·통계·직접인용·인용키·수식·표준약어)은 제외. 전체 근거는 `academic-tell-taxonomy.md`.

## 처리(FIX)
- **1 공허 메타:** "주목할 만한 점은 / 시사하는 바가 크다 / It is important to note" → 삭제, 내용 직진.
- **6 번역투:** "~를 통해→~하여", "~에 있어서→~에서", "~되어진다→~된다", "가지고 있다→형용사형", "그것은→생략".
- **7 리듬:** 문장 길이 변주(단문+장문 혼용, stdev>8), 종결 변주(~다/~었다/명사형), 단락 길이 단조 깨기.
- **8 접속사:** 문두 "또한·게다가·나아가" 70% 제거. "즉" ≤2회. "이는~" 메타 진입 삭제.
- **9 Hype:** "혁신적·놀랍게도·매우 중요한" 삭제 또는 구체 수치로.
- **10 서식:** em-dash→쉼표/괄호/분리, 과볼드·이모지 제거, 산문 속 불릿 산문화.
- **4a 중첩헤징:** 양태 2개+ 중첩 → 1개로.
- **5a 연쇄명사화:** 행위자 은폐 "~의 ~의 수행" → 동사구로.
- **11a 기계피동:** 전 문장 피동 일변도면 일부만 능동·연구자 주어.

## 플래그만(FLAG, 본문 수정 금지)
- **2 무출처 일반화:** "~로 알려져 있다 / 많은 연구가(인용0)" → `[저자 확인: 출처 필요]`.
- **3 환각·모호 인용:** 검증 불가 인용 → `[저자 확인: 인용 검증 필요]`. 절대 생성/삭제 금지.

## 보존(KEEP, 절대 건드리지 말 것 — `--academic`)
- **4b 단일 헤징:** "~로 보인다 / ~일 수 있다 / 제한된 자료에 근거하면" → 그대로.
- **5b 표준 명사화:** "타당성·신뢰도·정규성·상호작용효과·정규화" → 그대로.
- **11b 객관성 피동:** "측정되었다·통제되었다·표집되었다" → 그대로.
- IMRaD 구조, '~다' 평서체, 비축약, 정당한 논리 접속 1~2회, 인용·수치·수식.

## 자가검증 6점 (교정 후 5초)
1. 고유명사·숫자·인용·수식 보존? 2. 변경률 ≤30%? 3. 학술 규범(4b·5b·11b·구조) 훼손 0?
4. 평서체·격식 유지? 5. S1 잔여 0? 6. 없는 인용·근거를 만들지 않았는가?

등급: A(완벽) → D(S1 잔여 3+ 또는 >50% 변경 또는 인용 날조 의심 → 중단·사람 검토).
